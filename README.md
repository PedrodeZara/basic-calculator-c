# CLI Calculator (C)

Aplicação desenvolvida em linguagem C para realização de operações matemáticas básicas via terminal. O projeto tem como foco a prática de lógica de programação, controle de fluxo e manipulação de entrada do usuário.

---

## Funcionalidades

* Operações aritméticas:

  * Adição
  * Subtração
  * Multiplicação
  * Divisão
* Entrada de dados via terminal
* Execução em ambiente CLI

---

## Tecnologias utilizadas

* Linguagem C
* Compilador GCC

---

## Conceitos aplicados

* Estruturas condicionais (`if`, `else`)
* Estruturas de repetição (`while`, `for`)
* Entrada e saída de dados (`scanf`, `printf`)
* Lógica de programação

---

## Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/cli-calculator-c.git
```

2. Acesse a pasta do projeto:

```bash
cd cli-calculator-c
```

3. Compile o código:

```bash
gcc main.c -o calculadora
```

4. Execute o programa:

```bash
./calculadora
```

---

## Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de consolidar conceitos fundamentais da linguagem C, com foco em lógica de programação e interação via terminal.
